# MEP AI Platform — Demo Dashboards

Five interactive AI platform demos, each styled and branded for a specific MEP engineering consultancy in the Utrecht region.

## Demos

| Firm | Folder | Style |
|---|---|---|
| VIAC Adviseurs | `/viac` | Teal · Intelligent · Warm |
| DIA Groep | `/dia-groep` | Charcoal · Bold · Industrial |
| Feniks Installatie-adviseurs | `/feniks` | Amber · Experienced · Personal |
| DVTadvies | `/dvtadvies` | Forest Green · Ecological · Forward |
| Nieman Raadgevende Ingenieurs | `/nieman` | Navy · Authoritative · Refined |

## Features Demonstrated in Each Dashboard

1. **Project Overzicht** — Live project status cards with deadlines and team
2. **Rapport Schrijver** — AI-assisted technical report generation
3. **Excel Assistent** — Upload or paste Excel sheets, AI interprets and extends
4. **BENG Checker** — AI-assisted energy performance calculation tool
5. **Norm Monitor** — Real-time Dutch building regulation update tracker
6. **Kennisbank** — Semantic search across all past project knowledge

## Deployment

Each folder contains a standalone `index.html` — no build step needed.

Deploy with Vercel, Netlify, or GitHub Pages. Each firm gets its own URL path.

```
yourdomain.com/viac
yourdomain.com/dia-groep
yourdomain.com/feniks
yourdomain.com/dvtadvies
yourdomain.com/nieman
```
